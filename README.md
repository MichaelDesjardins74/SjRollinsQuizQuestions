# SJ Rollins Quiz Questions (Project Euler)

Solutions implemented in C# (.NET 10) as a console application.

Problems Included:
- #2 Even Fibonacci Numbers
- #18 Maximum Path Sum I
- #67 Maximum Path Sum II
- #81 Path Sum: Two Ways
- #82 Path Sum: Three Ways

Approach:
Dynamic programming used for path sum problems.
Iterative solution used for Fibonacci to avoid recursion overhead.

To Run:
Open solution in Visual Studio and press Run.
